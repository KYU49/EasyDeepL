# Easy DeepL
## 概要
段落にマウスを合わせると、DeepL翻訳(英->日)へのリンクが表示されるブックマークレットです。

主に論文を"Read Online"で読む場合を想定して作ってあります。  
※そのため、リンクの表示はHTMLの"p"タグ要素に限定しています。
